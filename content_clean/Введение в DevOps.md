Введение в DevOps

[[m0t1_Что такое DevOps]]
- Культура, а не инструмент.
- Цель: ускорить доставку ПО без потери качества.
- Принципы: CI/CD, IaC, мониторинг, автоматизация, коллаборация.
[[m0t2_Жизненный цикл разработки (SDLC)]]
- Planning → Development → Testing → Deployment → Monitoring → Feedback
[[m0t3_DevOps vs SRE vs Agile]]
- DevOps: объединяет dev и ops.
- SRE: инженерия надежности (Google).
- Agile: методология управления проектами.
[[m0t4_Непрерывная разработка и интеграция (CI & CD)]]
- Jenkins / GitLab CI / GitHub Actions
- Автоматизация тестов, сборки, деплоя
- Интеграция с командой: devs, testers, admins
[[p0_End-to-End Pipeline]]
- Настройка CI/CD для простого приложения
- Автоматический деплой на сервер
- Алерты при падении

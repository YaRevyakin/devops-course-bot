[[m13t1_Как ИИ меняет DevOps]]
Искусственный интеллект (ИИ) и машинное обучение (ML) постепенно входят в DevOps-практики, превращая реактивные процессы в проактивные и предиктивные. 
Это не замена инженеру, а усилитель его возможностей.

[[m13t2_Безопасность в работе с ИИ]]
- Принципы работы ИИ
- Промпт-инжиниринг
- Работа с корпоративными данными
- Корпоративные LLM
- Риски использования ИИ
[[m13t3_ИИ как ассистент разработчика]]
- Генерация, рефакторинг и документирование кода
- Запросы, которые понимает ИИ
- Генерация и отладка кода
- Рефакторинг с ИИ
- AI и тесты: минимальный юнит-пакет
- README и API-доки
[[m13t4_ИИ в инфраструктуре разработки]]
- ИИ в IDE: интеграция
- CI/CD с ИИ: от push до деплоя
- Code-review: нейросеть как ревьюер
- Лицензии и безопасный код с ИИ
- Нейросети и архитектура: визуализация и API
- Итоговая работа: пайплайн под ключ
[[m13t5_Поиск работы с помощью генеративного ИИ]]
- Формирование резюме и портфолио
- Подготовка к собеседованиям
- Видеолекции → обучение в своём темпе
[[m13t6_Нейросети для технических задач]]
- Решение задач аналитики с помощью ИИ
- SQL запросы и код с помощью ИИ (ChatGPT, GPT-4)
- Работа с Excel и Google Sheets через ИИ
- Возможности GPT-4 для продвинутой работы с данными

2. Какой инструмент обеспечивает автоматический failover для виртуального IP-адреса на Linux-серверах?
- [ ] A) HAProxy  
- [ ] B) Nginx  
- [ ] C) Keepalived  
- [ ] D) Prometheus

<details>
<summary>Показать ответ</summary>

C — Keepalived реализует протокол VRRP и позволяет двум (или более) серверам делить виртуальный IP. При сбое master-сервера backup автоматически берёт VIP.
</details>
4. Какой принцип лежит в основе горизонтального масштабирования?
- [ ] A) Увеличение мощности одного сервера  
- [ ] B) Добавление новых серверов в пул  
- [ ] C) Использование SSD вместо HDD  
- [ ] D) Сжатие сетевого трафика

<details>
<summary>Показать ответ</summary>

B — горизонтальное масштабирование означает добавление дополнительных узлов (серверов), а не усиление одного. Это повышает отказоустойчивость и гибкость.
</details>

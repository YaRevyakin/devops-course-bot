### 1. Что означает принцип «Confidentiality» в CIA-триаде?
- [ ] A) Данные доступны пользователям в любое время  
- [ ] B) Данные не могут быть изменены без авторизации  
- [ ] C) Данные доступны только авторизованным лицам  
- [ ] D) Система устойчива к DDoS-атакам

<details>
<summary>Показать ответ</summary>

**C** — Confidentiality (конфиденциальность) означает, что информация защищена от несанкционированного доступа. Например, пароли не должны быть видны посторонним.
</details>

### 2. Какой инструмент лучше всего подходит для сканирования уязвимостей в Docker-образах?
- [ ] A) Prometheus  
- [ ] B) Trivy  
- [ ] C) Grafana  
- [ ] D) HAProxy

<details>
<summary>Показать ответ</summary>

**B** — Trivy — лёгкий и быстрый сканер уязвимостей для контейнеров, файлов зависимостей и IaC. Широко используется в CI/CD.
</details>

### 3. Почему секреты (пароли, токены) нельзя хранить в Git?
- [ ] A) Git слишком медленный  
- [ ] B) Git не поддерживает шифрование  
- [ ] C) Любой с доступом к репозиторию увидит секреты, и их нельзя отозвать  
- [ ] D) Секреты занимают много места

<details>
<summary>Показать ответ</summary>

**C** — даже в приватном репозитории секреты могут быть скомпрометированы (например, при утечке доступа). Кроме того, они остаются в истории Git навсегда. Используй Vault или менеджеры секретов.
</details>

### 4. Какой принцип безопасности требует давать пользователю или сервису только минимально необходимые права?
- [ ] A) Defense in Depth  
- [ ] B) Zero Trust  
- [ ] C) Least Privilege  
- [ ] D) Fail Securely

<details>
<summary>Показать ответ</summary>

**C** — принцип наименьших привилегий (Least Privilege) минимизирует ущерб при компрометации: даже если злоумышленник получит доступ, его возможности будут ограничены.
</details>

### 5. Что делает команда `ufw default deny incoming`?
- [ ] A) Разрешает весь входящий трафик  
- [ ] B) Блокирует весь входящий трафик по умолчанию  
- [ ] C) Удаляет все правила firewall  
- [ ] D) Включает SSH-доступ

<details>
<summary>Показать ответ</summary>

**B** — эта команда устанавливает политику по умолчанию: **весь входящий трафик запрещён**, пока явно не разрешён (например, через `ufw allow 22`).
</details>

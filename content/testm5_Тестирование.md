### 1. Что такое SLO?
- [ ] A) Система логирования ошибок  
- [ ] B) Service Level Objective — целевой уровень надёжности сервиса  
- [ ] C) Программа для сбора метрик  
- [ ] D) Формат хранения логов

<details>
<summary>Показать ответ</summary>

**B** — SLO (Service Level Objective) — это измеримая цель надёжности, например: «99.9% запросов успешны за 30 дней».
</details>
---
### 2. Какой компонент используется для визуализации метрик в экосистеме Prometheus?
- [ ] A) Alertmanager  
- [ ] B) Node Exporter  
- [ ] C) Grafana  
- [ ] D) Fluentd

<details>
<summary>Показать ответ</summary>

**C** — Grafana — мощный инструмент для построения дашбордов на основе данных из Prometheus, Loki, Elasticsearch и других источников.
</details>
---
### 3. Чем Loki отличается от Elasticsearch при работе с логами?
- [ ] A) Loki индексирует всё содержимое логов, Elasticsearch — только метки  
- [ ] B) Loki не индексирует полное содержимое логов, а только метки — это дешевле  
- [ ] C) Loki работает только с Kubernetes  
- [ ] D) Elasticsearch не поддерживает JSON-логи

<details>
<summary>Показать ответ</summary>

**B** — Loki хранит логи как «blobs», а индексирует только **метки** (labels), что сильно снижает стоимость хранения по сравнению с полнотекстовым индексом Elasticsearch.
</details>

### 4. Какой инструмент лучше всего подходит для отслеживания исключений в коде приложения?
- [ ] A) Zabbix  
- [ ] B) Prometheus  
- [ ] C) Sentry  
- [ ] D) Grafana

<details>
<summary>Показать ответ</summary>

**C** — Sentry специализируется на сборе и анализе ошибок в приложениях: стектрейсы, контекст пользователя, частота возникновения.
</details>

### 5. Что делает node_exporter в стеке Prometheus?
- [ ] A) Визуализирует метрики  
- [ ] B) Отправляет алерты  
- [ ] C) Собирает системные метрики (CPU, RAM, диск) с хоста  
- [ ] D) Парсит логи Nginx

<details>
<summary>Показать ответ</summary>

**C** — node_exporter — это агент, который экспортирует метрики операционной системы (процессор, память, диск, сеть) в формате, понятном Prometheus.
</details>

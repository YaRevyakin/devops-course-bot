# Проект: Инфраструктура сайта

## [[Задача]]:
Разработать отказоустойчивую инфраструктуру для сайта в Yandex Cloud.

## [[Этапы]]:
1. Terraform → создать ВМ, сеть, балансировщик  
2. Ansible → настроить nginx, php, mysql  
3. CI/CD → автоматический деплой при изменении кода  
4. Мониторинг → Prometheus + Grafana + Alertmanager  
5. Логирование → Loki + Grafana  
6. Бэкапы → автоматическое копирование БД и файлов
